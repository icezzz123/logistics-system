## 运费计算公式验证

### 验证1：国内陆运
```
输入：重量=10kg, 运输方式=陆运(3), 国内
计算：10 × 10 × 1.0 × 1.0 = 100元
结果：100.00元 ✅
```

### 验证2：国际空运
```
输入：重量=5.5kg, 体积=0.2m³, 运输方式=空运(1), 国际
计算：
  重量运费 = 5.5 × 10 = 55元
  体积运费 = 0.2 × 200 = 40元
  基础运费 = max(55, 40) = 55元
  应用系数 = 55 × 1.5 × 1.5 = 123.75元
结果：123.75元 ✅
```

### 验证3：国际快递
```
输入：重量=2kg, 运输方式=快递(5), 国际
计算：2 × 10 × 2.0 × 1.5 = 60元
结果：60.00元 ✅
```

### 验证4：体积计费
```
输入：重量=1kg, 体积=0.5m³, 运输方式=陆运(3), 国内
计算：
  重量运费 = 1 × 10 = 10元
  体积运费 = 0.5 × 200 = 100元
  基础运费 = max(10, 100) = 100元
  应用系数 = 100 × 1.0 × 1.0 = 100元
结果：100.00元 ✅
```
