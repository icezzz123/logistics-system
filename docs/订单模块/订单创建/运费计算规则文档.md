# 运费计算规则文档

## 概述

本文档详细说明跨境物流系统的运费计算规则，包括基础运费、运输方式系数、距离系数、保险费等计算方法。

## 一、基础运费计算

### 1.1 计算原则
运费计算采用**重量和体积双重计费**原则，取两者计算结果的较大值作为基础运费。

### 1.2 重量运费
```
重量运费 = 货物重量(kg) × 单价(10元/kg)
```

**示例：**
- 货物重量：5kg
- 重量运费：5 × 10 = 50元

### 1.3 体积运费
```
体积运费 = 货物体积(m³) × 单价(200元/m³)
```

**示例：**
- 货物体积：0.5m³
- 体积运费：0.5 × 200 = 100元

### 1.4 基础运费确定
```
基础运费 = max(重量运费, 体积运费)
```

**示例：**
- 重量运费：50元
- 体积运费：100元
- 基础运费：100元（取较大值）

## 二、运输方式系数

不同的运输方式有不同的成本和时效，因此采用不同的系数。

### 2.1 系数表

| 运输方式 | 代码 | 系数 | 基础天数 | 特点 |
|---------|------|------|---------|------|
| 空运 | 1 | 1.5 | 3天 | 速度最快，成本较高 |
| 海运 | 2 | 0.5 | 30天 | 成本最低，时间最长 |
| 陆运 | 3 | 1.0 | 7天 | 标准运输，性价比高 |
| 铁路 | 4 | 0.8 | 10天 | 经济实惠，适合大宗 |
| 快递 | 5 | 2.0 | 2天 | 最快送达，成本最高 |

### 2.2 计算方法
```
运输运费 = 基础运费 × 运输方式系数
```

### 2.3 示例

#### 示例1：空运
- 基础运费：100元
- 空运系数：1.5
- 运输运费：100 × 1.5 = 150元

#### 示例2：海运
- 基础运费：100元
- 海运系数：0.5
- 运输运费：100 × 0.5 = 50元

#### 示例3：快递
- 基础运费：100元
- 快递系数：2.0
- 运输运费：100 × 2.0 = 200元

## 三、距离系数

根据运输距离（国内/国际）应用不同的系数。

### 3.1 系数规则

| 运输类型 | 判断条件 | 系数 | 说明 |
|---------|---------|------|------|
| 国内运输 | 发件国 = 收件国 | 1.0 | 不加价 |
| 国际运输 | 发件国 ≠ 收件国 | 1.5 | 加价50% |

### 3.2 计算方法
```
最终运费 = 运输运费 × 距离系数
```

### 3.3 示例

#### 示例1：国内运输
- 发件地：中国北京
- 收件地：中国上海
- 运输运费：100元
- 距离系数：1.0
- 最终运费：100 × 1.0 = 100元

#### 示例2：国际运输
- 发件地：中国深圳
- 收件地：美国纽约
- 运输运费：100元
- 距离系数：1.5
- 最终运费：100 × 1.5 = 150元

## 四、完整计算流程

### 4.1 计算步骤
```
步骤1：计算重量运费 = 重量 × 10
步骤2：计算体积运费 = 体积 × 200
步骤3：基础运费 = max(重量运费, 体积运费)
步骤4：应用运输方式系数
步骤5：应用距离系数
步骤6：四舍五入到小数点后2位
```

### 4.2 公式总结
```
最终运费 = max(重量×10, 体积×200) × 运输方式系数 × 距离系数
```

### 4.3 完整示例

**订单信息：**
- 货物重量：5kg
- 货物体积：0.2m³
- 运输方式：空运(1)
- 发件地：中国
- 收件地：美国

**计算过程：**
```
1. 重量运费 = 5 × 10 = 50元
2. 体积运费 = 0.2 × 200 = 40元
3. 基础运费 = max(50, 40) = 50元
4. 应用空运系数 = 50 × 1.5 = 75元
5. 应用国际系数 = 75 × 1.5 = 112.5元
6. 最终运费 = 112.50元
```

## 五、保险费计算

### 5.1 计算规则
```
保险费 = 保价金额 × 1%
```

### 5.2 约束条件
1. 保价金额必须大于0
2. 保价金额不能超过货物价值
3. 保险费四舍五入到小数点后2位

### 5.3 示例

#### 示例1：基本保险
- 保价金额：5000元
- 保险费：5000 × 0.01 = 50元

#### 示例2：高价值货物
- 保价金额：50000元
- 保险费：50000 × 0.01 = 500元

## 六、总费用计算

### 6.1 计算公式
```
订单总金额 = 运费 + 保险费
```

### 6.2 完整示例

**订单信息：**
- 货物重量：5kg
- 货物体积：0.2m³
- 运输方式：空运
- 发件地：中国
- 收件地：美国
- 是否保价：是
- 保价金额：5000元

**费用明细：**
```
运费计算：
  重量运费 = 5 × 10 = 50元
  体积运费 = 0.2 × 200 = 40元
  基础运费 = max(50, 40) = 50元
  空运系数 = 50 × 1.5 = 75元
  国际系数 = 75 × 1.5 = 112.5元
  运费小计 = 112.50元

保险费计算：
  保险费 = 5000 × 0.01 = 50元

总费用：
  订单总金额 = 112.50 + 50 = 162.50元
```

## 七、特殊情况处理

### 7.1 轻货（体积大重量轻）
当货物体积大但重量轻时，按体积计费。

**示例：**
- 货物：泡沫箱
- 重量：1kg
- 体积：0.5m³
- 重量运费：1 × 10 = 10元
- 体积运费：0.5 × 200 = 100元
- 基础运费：100元（按体积计费）

### 7.2 重货（重量大体积小）
当货物重量大但体积小时，按重量计费。

**示例：**
- 货物：金属零件
- 重量：50kg
- 体积：0.1m³
- 重量运费：50 × 10 = 500元
- 体积运费：0.1 × 200 = 20元
- 基础运费：500元（按重量计费）

### 7.3 未提供体积
如果未提供体积信息，仅按重量计费。

**示例：**
- 货物重量：10kg
- 货物体积：未提供
- 重量运费：10 × 10 = 100元
- 体积运费：0元
- 基础运费：100元

## 八、计费单位说明

### 8.1 单位规范
- **重量单位**：千克(kg)
- **体积单位**：立方米(m³)
- **金额单位**：人民币元(CNY)
- **时间单位**：天

### 8.2 精度要求
- 重量：支持小数点后2位
- 体积：支持小数点后2位
- 金额：四舍五入到小数点后2位

## 十、注意事项

1. 所有金额计算结果四舍五入到小数点后2位
2. 重量和体积必须为正数
3. 运输方式代码必须在1-5范围内
4. 国际运输判断基于发件国和收件国是否相同
5. 保险费仅在选择保价时计算
6. 系统会自动选择重量和体积运费中的较大值
7. 运费计算不包含关税和其他附加费用
8. 实际运费可能因特殊情况有所调整
