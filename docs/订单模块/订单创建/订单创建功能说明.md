# 订单创建功能说明

## 功能概述

订单创建功能是跨境物流系统的核心功能之一，支持用户创建物流订单，系统自动计算运费、保险费和预计送达时间。

## 功能特性

### 1. 订单信息采集
- **发件人信息**：姓名、电话、国家、省份、城市、详细地址、邮编
- **收件人信息**：姓名、电话、国家、省份、城市、详细地址、邮编
- **货物信息**：名称、类别、重量、体积、数量、价值
- **运输信息**：运输方式、服务类型
- **保险信息**：是否保价、保价金额
- **备注信息**：订单备注

### 2. 订单号生成
- **格式**：`ORD + 年月日时分秒(14位) + 客户ID后4位 + 毫秒(3位)`
- **示例**：`ORD202603041743550001114`
- **特点**：
  - 唯一性保证
  - 时间可读性
  - 客户关联性
  - 高并发支持

### 3. 运费计算

#### 3.1 基础运费计算
```
基础运费 = max(重量运费, 体积运费)
```

- **重量运费**：`重量(kg) × 10元/kg`
- **体积运费**：`体积(m³) × 200元/m³`
- 取两者较大值作为基础运费

#### 3.2 运输方式系数
| 运输方式 | 代码 | 系数 | 说明 |
|---------|------|------|------|
| 空运 | 1 | 1.5 | 速度快，成本高 |
| 海运 | 2 | 0.5 | 速度慢，成本低 |
| 陆运 | 3 | 1.0 | 标准运输 |
| 铁路 | 4 | 0.8 | 经济实惠 |
| 快递 | 5 | 2.0 | 最快速度 |

#### 3.3 距离系数
- **国内运输**：系数 1.0（不加价）
- **国际运输**：系数 1.5（加价50%）

#### 3.4 运费计算公式
```
最终运费 = 基础运费 × 运输方式系数 × 距离系数
```

### 4. 保险费计算
```
保险费 = 保价金额 × 1%
```

- 最低保险费：根据保价金额计算
- 保价金额不能超过货物价值

### 5. 预计送达时间

#### 5.1 基础天数
| 运输方式 | 国内基础天数 | 国际额外天数 |
|---------|-------------|-------------|
| 空运 | 3天 | +5天 |
| 海运 | 30天 | +5天 |
| 陆运 | 7天 | +5天 |
| 铁路 | 10天 | +5天 |
| 快递 | 2天 | +5天 |

#### 5.2 计算规则
```
预计天数 = 基础天数 + (国际运输 ? 5 : 0)
```

## 参数验证规则

### 1. 发件人信息验证
- 姓名：必填，最大50字符
- 电话：必填，最大20字符
- 国家：必填，最大50字符
- 城市：必填，最大50字符
- 地址：必填，最大255字符
- 省份：可选，最大50字符
- 邮编：可选，最大20字符

### 2. 收件人信息验证
- 规则同发件人信息

### 3. 货物信息验证
- 名称：必填，最大100字符
- 重量：必填，大于0，不超过10000kg
- 体积：可选，不能为负数，不超过1000m³
- 数量：可选，最小1件
- 价值：可选，不能为负数
- 类别：可选，最大50字符

### 4. 运输信息验证
- 运输方式：必填，1-5之间的整数
- 服务类型：可选，最大20字符

### 5. 保险信息验证
- 是否保价：0或1
- 保价金额：如果保价，必须大于0
- 保价金额不能超过货物价值

## 业务规则

### 1. 订单状态
- 新创建的订单状态为：**待处理(1)**
- 支付状态为：**未支付(unpaid)**

### 2. 货物数量
- 如果未指定数量，默认为1件

### 3. 币种
- 默认币种：人民币(CNY)

### 4. 时间记录
- 订单时间：创建时的Unix时间戳
- 创建时间：自动记录
- 修改时间：自动更新

## 使用场景

### 场景1：国内快递
- 发件地：北京
- 收件地：上海
- 货物：文件，1kg
- 运输方式：快递
- 运费计算：1kg × 10 × 2.0 = 20元

### 场景2：国际空运
- 发件地：中国深圳
- 收件地：美国纽约
- 货物：电子产品，5kg
- 运输方式：空运
- 保价：5000元
- 运费计算：5kg × 10 × 1.5 × 1.5 = 112.5元
- 保险费：5000 × 1% = 50元
- 总费用：162.5元

### 场景3：体积计费
- 货物：泡沫箱，1kg，0.5m³
- 重量运费：1kg × 10 = 10元
- 体积运费：0.5m³ × 200 = 100元
- 取较大值：100元

## 错误处理

### 常见错误
1. **参数错误**：缺少必填字段或格式不正确
2. **重量超限**：货物重量超过10000kg
3. **体积超限**：货物体积超过1000m³
4. **保价错误**：保价金额超过货物价值
5. **运输方式无效**：运输方式代码不在1-5范围内

### 错误响应示例
```json
{
  "code": 400,
  "message": "货物重量必须大于0"
}
```

## 注意事项

1. 所有金额单位为人民币（元），保留2位小数
2. 重量单位为千克（kg）
3. 体积单位为立方米（m³）
4. 时间使用Unix时间戳（秒）
5. 订单创建后不可修改基本信息，只能修改状态
6. 国际运输需要提供完整的地址信息
7. 保价金额建议不超过货物实际价值
8. 订单号全局唯一，不可重复
