### 测试用例1：默认分页查询

**测试目的**：验证默认分页参数的正确性

2. 访问 `GET /api/orders`（不带任何参数）

**预期结果**：
- 返回第1页数据
- 每页10条记录
- 返回所有5个订单
- 按时间倒序排列

**实际结果**：
```json
{

  "page_size": 10,
  "pages": 1
}
```

**测试结果**：✅ 通过
