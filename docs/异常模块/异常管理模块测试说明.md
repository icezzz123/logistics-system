# 异常管理模块测试说明

## 测试目标
验证异常模块的以下能力：
- 异常创建
- 异常列表查询
- 异常详情查询
- 异常分配
- 异常处理
- 异常关闭
- 异常统计分析
- 订单状态与异常流程联动

---

## 前置条件
- 服务已启动
- 已存在可登录账号
- 已存在测试订单
- 请求头中已带有效 JWT Token

---

## 推荐测试用例

### 用例1：创建异常
**请求**: `POST /api/exceptions`

**验证点**:
- 返回异常单号
- 状态为“待处理”
- 订单状态变为“异常”

### 用例2：查询异常列表
**请求**: `GET /api/exceptions?page=1&page_size=10&order_no=xxx`

**验证点**:
- 支持分页
- 支持按订单号筛选
- 返回列表总数、状态、类型信息

### 用例3：查询异常详情
**请求**: `GET /api/exceptions/:id`

**验证点**:
- 返回完整异常信息
- 包含订单、站点、上报人、处理人等关联字段

### 用例4：分配处理人
**请求**: `PUT /api/exceptions/:id/assign`

**验证点**:
- `handler_id` 更新成功
- 已关闭异常不能分配

### 用例5：处理异常
**请求**: `PUT /api/exceptions/:id/process`

**验证点**:
- 状态可更新为处理中或已解决
- 方案、结果、赔付金额更新成功

### 用例6：关闭异常并恢复订单状态
**请求**: `PUT /api/exceptions/:id/close`

**验证点**:
- 异常状态变为已关闭
- `close_time` 有值
- 如果传入 `resume_status`，订单状态应恢复到目标状态

### 用例7：异常统计分析
**请求**: `GET /api/exceptions/stats`

**验证点**:
- 返回汇总数据
- 返回按类型、状态、站点、日期的统计结果

## PowerShell联调示例

### 1. 登录获取 Token
```powershell
$loginBody = @{
    username = 'admin'
    password = 'Pass1234'
} | ConvertTo-Json

$loginResp = Invoke-RestMethod -Method Post `
    -Uri 'http://127.0.0.1:8080/api/auth/login' `
    -ContentType 'application/json' `
    -Body $loginBody

$token = $loginResp.data.token
$headers = @{ Authorization = "Bearer $token" }
```

### 2. 创建异常
```powershell
$body = @{
    order_id = 31
    type = 3
    station_id = 1
    description = '运输节点长时间停滞，疑似延误'
    images = @('https://example.com/img1.jpg')
    remark = '异常联调测试'
} | ConvertTo-Json

Invoke-RestMethod -Method Post `
    -Uri 'http://127.0.0.1:8080/api/exceptions' `
    -Headers $headers `
    -ContentType 'application/json' `
    -Body $body
```

### 3. 查询异常列表
```powershell
Invoke-RestMethod -Method Get `
    -Uri 'http://127.0.0.1:8080/api/exceptions?page=1&page_size=10' `
    -Headers $headers
```

### 4. 分配处理人
```powershell
$body = @{
    handler_id = 5
    remark = '分配处理人'
} | ConvertTo-Json

Invoke-RestMethod -Method Put `
    -Uri 'http://127.0.0.1:8080/api/exceptions/1/assign' `
    -Headers $headers `
    -ContentType 'application/json' `
    -Body $body
```

### 5. 处理异常
```powershell
$body = @{
    status = 3
    solution = '重新安排运输'
    result = '异常已解决'
    compensate_amount = 25
    remark = '处理完成'
} | ConvertTo-Json

Invoke-RestMethod -Method Put `
    -Uri 'http://127.0.0.1:8080/api/exceptions/1/process' `
    -Headers $headers `
    -ContentType 'application/json' `
    -Body $body
```

### 6. 关闭异常
```powershell
$body = @{
    resume_status = 5
    remark = '关闭异常并恢复订单状态'
    result = '联调关闭成功'
} | ConvertTo-Json

Invoke-RestMethod -Method Put `
    -Uri 'http://127.0.0.1:8080/api/exceptions/1/close' `
    -Headers $headers `
    -ContentType 'application/json' `
    -Body $body
```

### 7. 查询异常统计
```powershell
Invoke-RestMethod -Method Get `
    -Uri 'http://127.0.0.1:8080/api/exceptions/stats' `
    -Headers $headers
```
