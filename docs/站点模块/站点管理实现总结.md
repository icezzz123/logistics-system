## 实现内容

### 1. 服务层 (services/station_service.go)

#### 实现的方法

| 方法名 | 功能 | 说明 |
|--------|------|------|
| `CreateStation` | 创建站点 | 验证站点编码唯一性、管理员角色 |
| `GetStationByID` | 根据ID获取站点 | 单个站点查询 |
| `GetStationList` | 获取站点列表 | 支持分页、筛选、搜索 |
| `UpdateStation` | 更新站点 | 支持部分字段更新 |
| `DeleteStation` | 删除站点 | 检查订单关联 |

#### 业务逻辑特点

1. **唯一性验证**
   - 站点编码必须唯一
   - 创建时自动检查

2. **管理员验证**
   - 如果指定管理员，验证用户存在且角色为5（站点管理员）
   - 角色验证确保权限正确

3. **删除保护**
   - 检查站点是否被订单引用
   - 防止删除有关联数据的站点

4. **灵活查询**
   - 支持按类型、国家、城市、状态筛选
   - 支持关键词搜索（站点编码、名称、地址）
   - 支持分页

5. **部分更新**
   - 只更新提供的字段
   - 未提供的字段保持不变

### 3. 路由配置 (routes/routes.go)

#### 路由分组

**公开查询路由** (需要认证)
```go
/api/stations              GET    获取站点列表
/api/stations/:id          GET    获取站点详情
```

**管理员路由** (需要管理员权限)
```go
/api/admin/stations        POST   创建站点
/api/admin/stations/:id    PUT    更新站点
/api/admin/stations/:id    DELETE 删除站点
```

#### 权限控制

- 查询操作：所有已认证用户
- 创建/更新/删除：仅管理员 (role=7)

## 使用方法

### 1. 启动服务器

```bash
go run main.go
```

服务将在 `http://localhost:8080` 启动。

### 2. 导入 Postman 集合

1. 打开 Postman
2. 点击 Import
3. 选择 `logistics-system.postman_collection.json`
4. 导入成功

### 3. 运行测试

**方式1：运行整个集合**
1. 在 Postman 中选择集合
2. 点击 "Run" 按钮
3. 查看测试结果

**方式2：单独运行请求**
1. 先运行 "Authentication" → "Login" 获取 token
2. 然后运行 "Station Management" 下的各个请求

### 4. 手动测试

```bash
# 1. 登录
curl -X POST http://localhost:8080/api/auth/login \
  -H "Content-Type: application/json" \
  -d '{"username":"admin","password":"123456"}'

# 2. 创建站点
curl -X POST http://localhost:8080/api/admin/stations \
  -H "Authorization: Bearer {token}" \
  -H "Content-Type: application/json" \
  -d '{
    "station_code": "STA001",
    "name": "深圳始发站",
    "type": 1,
    "country": "中国",
    "city": "深圳市",
    "address": "南山区科技园",
    "capacity": 5000
  }'

# 3. 查询站点列表
curl -X GET "http://localhost:8080/api/stations?page=1&page_size=10" \
  -H "Authorization: Bearer {token}"
```

## 数据库表结构

### stations 表

| 字段名 | 类型 | 说明 | 约束 |
|--------|------|------|------|
| id | BIGINT | 主键 | AUTO_INCREMENT |
| station_code | VARCHAR(50) | 站点编码 | UNIQUE, NOT NULL |
| name | VARCHAR(100) | 站点名称 | NOT NULL |
| type | INT | 站点类型 | NOT NULL |
| country | VARCHAR(50) | 国家 | NOT NULL, INDEX |
| province | VARCHAR(50) | 省份 | INDEX |
| city | VARCHAR(50) | 城市 | NOT NULL, INDEX |
| address | VARCHAR(255) | 详细地址 | NOT NULL |
| latitude | DECIMAL(10,6) | 纬度 | - |
| longitude | DECIMAL(10,6) | 经度 | - |
| manager_id | BIGINT | 管理员ID | INDEX, DEFAULT 0 |
| capacity | INT | 容量 | NOT NULL, DEFAULT 1000 |
| contact_name | VARCHAR(50) | 联系人姓名 | - |
| contact_phone | VARCHAR(20) | 联系电话 | - |
| working_hours | VARCHAR(100) | 工作时间 | - |
| status | INT | 状态 | DEFAULT 1 |
| remark | TEXT | 备注 | - |
| ctime | BIGINT | 创建时间 | NOT NULL |
| mtime | BIGINT | 修改时间 | NOT NULL |
